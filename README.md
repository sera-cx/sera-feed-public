Generated feed JSON for the **sera.money** app. Built and pushed automatically — do not edit by hand; the source pipeline lives in a private repo.

- `latest.json` — current feed (brief + categories + items)
- `YYYY-MM-DD.json` — daily archives

Served via GitHub Pages: `https://sera-cx.github.io/sera-feed-public/latest.json`

Content is information, not investment advice.
